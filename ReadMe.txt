Tout d'abord lorsque l'on lance le jeu on nous demande de choisir notre classe parmis celles disponibles. (La classe Archer à des stats plus puissante pour avancer plus facilement)
Il suffit d'écrire le nom de la classe souhaitée donc il faut écrire soit:
"Archer"
"Assassin"
"Barbare"
"Sorcier"

Ensuite on nous demander d'entrer un nom.
Il suffit d'écrire le nom que l'on souhaite

Le jeu commence

On a des informations qui s'affiche, il y a le nombre de pièces dans le donjon, le nombre d'ennemis dans la pièce actuelle ainsi que la liste des ennemis présent dans cette pièce.

On nous demande si l'on veut attaquer ou afficher l'inventaire.

Pour attaquer il suffit d'écrire "attaquer" et ainsi nous allons attaquer le premier ennemi de la liste.
Le combat se déroule au tour par tour et l'ordre d'attaque entre le joueur et l'ennemi est déterminer par rapport à une stat de vitesse. Celui qui a la plus élevée attaque en premier.
Si l'on tue l'ennemi le jeu nous afficher qu'il ai mort sinon il affiche la vie restante de l'ennemi ainsi que celle du joueur après que chacun est attaquer.
Lorsque l'ennemi est mort s'il en reste d'autres dans la pièce le jeu nous affiche la liste des ennemis restant et l'étape du combat recommence. Sinon si la salle est terminer un item est généré aléatoirement et placé au sol j'expliquerait cette partie ensuite.
Si l'ennemi n'est pas mort directement un nouveau tour se lance et on peut donc rechoisir entre attaquer et afficher l'inventaire et si l'on chosit attaquer on va a nouveau attaquer le meme ennemi.

 Pour afficher l'inventaire il suffit d'écrire "afficher inventaire".
 Si l'inventaire est vide le jeu affiche inventaire vide.
 Sinon le jeu affiche la liste des objets présents dans notre inventaire.
 Il nous demande ensuite si l'on veut équiper un objet, consommer un element ou quitter l'inventaire.
 
 Pour équiper un objet il suffit d'écrire "equiper un objet" ensuite le jeu nous demande le nom de l'objet que l'on souhaite equiper.
 Il suffit d'écrire le nom complet de la pièce d'armure ou de l'épeé que l'on souhaite.
 Ensuite le jeu nous dit que notre objet a été équipé ou a remplacer un objet du même type qui était deja équipé. Il affiche ensuite nos stats qui ont changées.
 
Pour consommer un element il suffit d'écrire "consommer un element" ensuite le jeu nous demander le nom de l'élément à consommer.
Il suffit d'écrire le nom de l'élément à consommer, "pain" ou "potion de soin". Les deux nous rendent de la vie.

Pour quitter l'inventaire il suffit d'écrire "quitter inventaire". 

Le jeu reprend ensuite normalement.

Lorsque l'on termine une pièce un objet apparait au sol et le jeu nous propose de le rammasser.
Pour le rammasser il suffit d'écrire "oui" sinon il faut écrire "non".

Le jeu nous fait ensuite automatiquement avancer dans une autre pièce.

Lorsque le donjon est terminé le jeu nous emmène directement dans un nouveau.

Le but du jeu est d'aller le plus loin possible donc le seul moyen de terminer une partie est de mourir.
Lorsque le joueur meurt, le jeu affiche ces stats lors de la partie ainsi qu'un score qu'il pourra essayer de battre à la partie suivante.

Pour finir il nous demande si l'on veut refaire une partie.
Pour rejouer il suffit d'écrire "oui" sinon il suffit de d'écrire n'importe quoi d'autre pour que le jeu se ferme.

Si l'on choisit de rejouer une partie se relance et on repart au choix de la classe.
